### Convention
1. 기능별로 폴더를 분류하고 관련 소스는 해당 폴더에 정리하기
2. 파일 경로를 지정할 때에는 상대경로로 작성 ('./file.f' (O), 'C://file.f' (X))
3. 파일 용량이 크거나 민감 정보(key file), 라이브러리 raw source는 .gitignore에 추가하기
